# HubDev AI Landing Page

This project serves as the landing page and documentation hub for the HubDev AI ecosystem, featuring a "Liquid Desktop" interface with floating, draggable elements.

## Ecosystem & Branding

The following naming conventions are locked for the project:

### Untrusted<T> (formerly AILang)
*   **Language (Marketing/Docs Title):** `Untrusted<T>`
*   **Compiler/CLI Brand:** `sec4Audit`
*   **Policy Tools:**
    *   `sec4 audit`
    *   `sec4 explain`
    *   `sec4 gate`
*   **GitHub Repo:** `https://github.com/HubDev-AI/untrusted-compiler`
*   **File Extension:** `.ut` (or `.un`)
*   **Package Namespace:** `ut/std`, `ut/http`, `ut/sec`

## Development

To run the landing page locally:

```bash
npm install
npm run dev
```

## Features

*   **Liquid Desktop**: Glassmorphic, animated background.
*   **Draggable Boxes**: Project cards are floatable and interactive.
*   **Responsive**: Designed to work on various screen sizes.
